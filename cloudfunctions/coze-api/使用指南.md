# Coze API 云函数 - 使用指南

## 部署步骤

### 1. 上传云函数

在微信开发者工具中：
1. 展开 `cloudfunctions/` 
2. 右键 `coze-api` → **上传并部署：云端安装依赖**
3. 等待上传完成

### 2. 开启 HTTP 访问（可选）

如需用 `wx.request` 调用，需开启 HTTP 访问：
1. 点顶部工具栏 **「云开发」** ☁️ 按钮
2. 进入「**云函数**」→ 选择 `coze-api`
3. 点「**HTTP访问**」→ 开启
4. 复制生成的 URL（如 `https://xxx.service.tcloudbase.com/coze-api`）

---

## 调用方式

### 方式一：wx.cloud.callFunction（推荐，免配置）

```javascript
// 发送消息
wx.cloud.callFunction({
  name: 'coze-api',
  data: {
    action: 'chat',
    message: '分析2025年全球光伏市场',
    userId: '用户openid（可选）',
  },
  success: (res) => {
    if (res.result.code === 0) {
      console.log('AI回复:', res.result.data.reply);
    } else {
      console.error('错误:', res.result.message);
    }
  },
  fail: (err) => {
    console.error('调用失败:', err);
  },
});
```

### 方式二：wx.request（需开启 HTTP 访问）

```javascript
const API_URL = 'https://xxx.service.tcloudbase.com/coze-api'; // 替换为实际地址

wx.request({
  url: API_URL,
  method: 'POST',
  header: { 'Content-Type': 'application/json' },
  data: {
    message: '分析2025年全球光伏市场',
    userId: '用户openid（可选）',
  },
  success: (res) => {
    if (res.data.code === 0) {
      console.log('AI回复:', res.data.data.reply);
    }
  },
});
```

### 方式三：健康检查

```javascript
wx.cloud.callFunction({
  name: 'coze-api',
  data: { action: 'ping' },
  success: (res) => console.log(res), // { code: 0, message: 'pong' }
});
```

## 返回数据格式

### 成功
```json
{
  "code": 0,
  "message": "成功",
  "data": {
    "reply": "## 📊 市场分析\n\n...（Markdown 格式）",
    "conversation_id": "7488314421",
    "chat_id": "7488314421"
  }
}
```

### 失败
```json
{ "code": 504, "message": "AI 响应超时，请稍后重试", "data": null }
```

### 错误码说明

| code | 说明 |
|------|------|
| 0 | 成功 |
| 400 | 参数错误（缺消息/消息过长） |
| 502 | AI 服务不可用 |
| 504 | AI 响应超时 |
| 500 | 服务器内部错误 |

## 完整页面示例

以下是一个完整的 AI 对话页面代码，可直接在小程序中使用：

### chat.js
```javascript
Page({
  data: { messages: [], inputText: '', isLoading: false, userId: '' },

  onLoad() {
    const app = getApp();
    this.setData({ userId: app?.globalData?.openid || `user_${Date.now()}` });
  },

  onInput(e) {
    this.setData({ inputText: e.detail.value });
  },

  sendMessage() {
    const text = this.data.inputText.trim();
    if (!text || this.data.isLoading) return;
    this.setData({ inputText: '' });
    this.addMessage('user', text);
    this.setData({ isLoading: true });
    this.callAI(text);
  },

  callAI(message) {
    wx.cloud.callFunction({
      name: 'coze-api',
      data: { action: 'chat', message, userId: this.data.userId },
      success: (res) => {
        if (res.result.code === 0) {
          this.addMessage('assistant', res.result.data.reply);
        } else {
          this.addMessage('assistant', `⚠️ ${res.result.message}`, true);
        }
      },
      fail: () => {
        this.addMessage('assistant', '⚠️ 网络连接失败', true);
      },
      complete: () => {
        this.setData({ isLoading: false });
      },
    });
  },

  addMessage(role, content, isError = false) {
    this.setData({
      messages: [...this.data.messages, { role, content, time: new Date().toLocaleTimeString(), isError }],
    });
  },
});
```
